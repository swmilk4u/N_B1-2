## 3. 작업 전략 (제작 파이프라인)

```
1. 기획  →  2. 스토리보드  →  3. 소스 생성  →  4. 편집·통합  →  5. 출품
```

### 단계별 체크리스트

- [ ] **기획** : 주제 선정 (공모 주제 3개 중 선택 또는 혼합), 브랜드/캠페인 메시지 정의
- [ ] **스토리보드** : 씬 구성 작성, 프롬프트 수정 전/후 기록 (최소 1개 씬)
- [ ] **이미지 생성** : 무료 AI 도구로 씬별 키비주얼 생성
- [ ] **영상 변환** : 이미지 → 모션 변환 또는 영상 직접 생성
- [ ] **오디오** : 공식 지정 음원 다운로드 + 필요 시 효과음/보이스 생성
- [ ] **편집** : 컷 편집, 자막, 브랜드 인지 장치(마지막 3~5초) 삽입, 공식 AI 워터마크 표기
- [ ] **검토** : 길이(20~60초), 해상도(1080×1920, 16:9), 해시태그, 워터마크 최종 확인
- [ ] **SNS 업로드** : 유튜브/인스타그램/틱톡 중 택1, 해시태그 등록, URL 메모
- [ ] **접수** : 홈페이지에서 URL 제출 (7.1~7.15)

## 7. 씬별 스토리보드 필수 항목

모든 씬에 아래 항목을 누락 없이 기재합니다.

- 씬 번호 / 씬 길이(초)
- 목표 메시지 (한 문장)
- 화면 구성 (구도 / 피사체 / 배경 / 텍스트 유무)
- 내레이션 또는 화면 카피
- 사용 도구 및 목적
- 입력 프롬프트 (원문) + 출력 결과 요약 (한 줄)
- 생성 결과 파일명

> 최소 **1개 씬**에서 프롬프트 수정 전/후 비교 및 수정 이유 기록 필수

## 8. 유의 사항

- 유료 AI 플랜 사용 시 **부정행위**로 간주되어 심사 제외
- 딥페이크·혐오·선정성·폭력 등 유해 콘텐츠 생성 **금지**
- 접수 완료 후 작품 변경 **불가** → 업로드 전 최종 검토 필수
- 수상 작품은 동영상 **원본 파일 제출** 요청될 수 있으므로 원본 보관
- SNS 영상 등록 날짜는 **접수일(7.1) 이전**이어야 함
