| 타겟 | 생성형 AI 창작 도구를 사용하는 대학생·크리에이터·일반인 (20~40대) |

## 2. 캠페인 목표 및 서사 구조

**광고 목적** : AI 저작권 인식 제고 캠페인 (인지 + 태도 변화)

**서사 구조 (기승전결)**

```
[기]  누군가의 창작물이 세상에 나온다
[승]  그 수많은 창작물이 AI를 학습시켰다
[전]  이제 당신도 AI로 창작하는 사람이다
[결]  만들기 전에, 한 번 더 — 출처·허락·존중
```

## 5. 씬별 스토리보드

### 씬 1 — 오프닝 : 창작자의 손 (0~7초, 7초)

| 필드 | 내용 |
|------|------|
| 씬 번호 | 1 |
| 씬 길이 | 7초 |
| 목표 메시지 | 사람의 손으로 탄생하는 창작물의 가치를 시각적으로 각인시킨다 |
| 화면 구성 | 클로즈업 / 손가락 끝·붓·키보드 / 어두운 스튜디오 배경 / 텍스트 없음 |
| 내레이션 / 카피 | (없음 — 음악과 영상만) |
| 사용 도구 | Ideogram (이미지), Runway Gen-3 (모션 변환) |
| 사용 목적 | 화가의 손·음악가의 건반·작가의 타이핑을 각각 키비주얼로 생성 후 영상으로 변환 |
| 입력 프롬프트 | `close-up of a painter's hand holding a brush, dramatic studio lighting, dark background, cinematic, 4k, minimal` |
| 출력 결과 요약 | 붓을 쥔 손 클로즈업, 극적 조명의 시네마틱 분위기 확보 |
| 생성 파일명 | `scene01_hand_painter.png` / `scene01_motion.mp4` |

---

### 씬 2 — 기승 : AI 학습 시각화 (7~17초, 10초)

| 필드 | 내용 |
|------|------|
| 씬 번호 | 2 |
| 씬 길이 | 10초 |
| 목표 메시지 | 수많은 창작물이 AI를 만들었다는 사실을 시각적으로 전달한다 |
| 화면 구성 | 광각·하이앵글 / 화면을 가득 채우는 이미지·텍스트·악보·그림들이 하나의 AI 아이콘으로 수렴하는 추상 시각화 / 어두운 우주 느낌 배경 / 텍스트 있음 |
| 내레이션 / 카피 | `"이 모든 창작물이..."` (3초 후) `"AI를 만들었습니다."` |
| 사용 도구 | Ideogram (키비주얼), Runway Gen-3 (수렴 모션 효과), ElevenLabs (TTS 내레이션) |
| 사용 목적 | 수많은 아이콘이 중앙으로 모이는 추상 애니메이션 생성으로 AI 학습 개념 시각화 |
| 입력 프롬프트 | `abstract visualization of thousands of artworks, music notes, text fragments converging into a glowing AI brain icon, dark space background, blue and white particles, cinematic 4k` |
| 출력 결과 요약 | 파란 파티클이 중앙 AI 아이콘으로 수렴하는 추상 시각 확보 |
| 생성 파일명 | `scene02_ai_learning.png` / `scene02_motion.mp4` / `scene02_narration.mp3` |

### 씬 3 — 전환 : 당신도 이제 창작자 (17~25초, 8초)

| 필드 | 내용 |
|------|------|
| 씬 번호 | 3 |
| 씬 길이 | 8초 |
| 목표 메시지 | AI 도구를 사용하는 사람도 창작 행위의 주체임을 인식시킨다 |
| 화면 구성 | 미디엄 샷 / 노트북 화면 앞에 앉은 인물 실루엣, 화면에서 이미지가 생성되는 장면 / 밝은 자연광 배경으로 전환 / 텍스트 있음 |
| 내레이션 / 카피 | `"이제, 당신도 창작자입니다."` |
| 사용 도구 | Ideogram (이미지), Runway Gen-3 (모션), ElevenLabs (TTS) |
| 사용 목적 | 인물 실루엣으로 시청자가 자신을 투영할 수 있게 하고, 화면 전환으로 희망적 분위기 연출 |
| 입력 프롬프트 | `silhouette of a person sitting at a laptop, glowing screen showing AI-generated art, bright natural light from window, warm tones, cinematic minimal` |
| 출력 결과 요약 | 따뜻한 조명 아래 노트북 앞 실루엣 인물, 전환 분위기 연출 성공 |
| 생성 파일명 | `scene03_creator.png` / `scene03_motion.mp4` / `scene03_narration.mp3` |

### 씬 4 — 결 : 핵심 메시지 전달 (25~35초, 10초)

| 필드 | 내용 |
|------|------|
| 씬 번호 | 4 |
| 씬 길이 | 10초 |
| 목표 메시지 | 구체적인 행동 지침(출처 표기·허락·존중)을 명확하게 전달한다 |
| 화면 구성 | 센터 프레임 / 원작자와 AI 사용자가 나란히 서 있는 미니멀 일러스트 스타일 / 흰 배경 / 텍스트 강조 있음 |
| 내레이션 / 카피 | `"출처를 밝히고,"` → `"허락을 구하고,"` → `"창작자를 존중합니다."` |
| 사용 도구 | Ideogram (미니멀 일러스트 이미지), Runway Gen-3 (텍스트 페이드인 모션), ElevenLabs (TTS) |
| 사용 목적 | 미니멀 일러스트로 두 존재의 공존을 표현, 3단계 행동 지침을 순차 텍스트로 강조 |
| 입력 프롬프트 | `minimal flat illustration, two figures side by side, one is an artist holding a paintbrush, one is using a laptop, white background, pastel blue and warm yellow color palette, simple clean design` |
| 출력 결과 요약 | 파스텔 톤 미니멀 일러스트, 두 인물의 공존 이미지 확보 |
| 생성 파일명 | `scene04_coexist.png` / `scene04_motion.mp4` / `scene04_narration.mp3` |
